# Meeting Intelligence Hub (SmartMinutes)

A full-stack AI-powered application designed to process meeting transcripts, extract structured insights (Decisions and Action Items), and provide a conversational query interface to chat with your transcripts.

## Features

- **Automated Extraction:** Upload `.txt` or `.vtt` meeting transcripts to automatically identify and extract key decisions and action items (including assignees and deadlines).
- **Contextual Query Chatbot:** Ask natural language questions about the meeting, and the AI will provide accurate, context-aware answers based solely on the transcript.
- **Export Capabilities:** Export your structured meeting minutes and reports to PDF.
- **Modern UI/UX:** A clean, responsive interface built with Next.js and React.

## Technology Stack

### Frontend
- **Framework:** Next.js & React
- **Libraries:** `jsPDF` (for exports), `lucide-react` (icons), `react-markdown` (for rendering AI responses).

### Backend
- **Framework:** FastAPI (Python)
- **AI Integration:** Google Generative AI API (Gemini 2.5 Flash model)
- **Environment Management:** `python-dotenv`

## Getting Started

### Prerequisites
- Node.js (v18+)
- Python (v3.9+)
- A [Google Gemini API Key](https://aistudio.google.com/app/apikey)

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Meenakshi-prog15/Meeting-Intelligence-Hub.git
   cd Meeting-Intelligence-Hub
   ```

2. **Set up the Backend:**
   ```bash
   cd backend
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   pip install -r requirements.txt
   ```

3. **Set up the Frontend:**
   ```bash
   cd ../frontend
   npm install
   ```

4. **Environment Variables:**
   Create a `.env.local` file in the `frontend/` directory with your Google Gemini API Key:
   ```env
   GOOGLE_GENERATIVE_AI_API_KEY=your_api_key_here
   ```

### Running the Application

1. **Start the Backend Server:**
   ```bash
   # From the backend directory
   python main.py
   # The API will be available at http://localhost:8000
   ```

2. **Start the Frontend Development Server:**
   ```bash
   # From the frontend directory
   npm run dev
   # The web app will be available at http://localhost:3001
   ```

## Usage

1. Open `http://localhost:3001` in your browser.
2. Upload a valid meeting transcript text file.
3. Review the extracted Decisions and Action Items generated by the AI.
4. Use the contextual chatbot interface to ask specific questions about the meeting.
5. Export the summarized minutes as a PDF document directly from the interface.
